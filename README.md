# 🌐 [MultiverseHub](https://multiversehub.onrender.com/)

**MultiverseHub** is a full-stack platform that brings together your favorite digital content — all in one place.  
No more juggling between apps for news, cricket, entertainment, or games.  
With additional features like a favorite section, global search, user authentication, theme preferences, and a dynamic UI/UX, **MultiverseHub** is your one-stop destination.

---

## ✨ Features

- 🔐 **Authentication** — JWT-based login, signup, logout, and password update
- 🎨 **Light/Dark Mode** — Toggle and save theme preference to backend
- 🧠 **Global Search** — Search across all content sections (News, Cricket, Entertainment, Games)
- ⭐ **Favorites Section** — Save your favorite items from any section, view/remove them later
- 👤 **User Profile** — Personalized greeting, avatar, profile info, update password
- 📺 **Entertainment Section** — Browse/search Movies and TV Series with categories and detailed view
- 📰 **News Section** — Latest news by category, with search and weather lookup
- 🏏 **Cricket Section** — Recent matches, latest cricket news, player rankings, points table, and player info
- 🎮 **Games Section** — Discover games by category, filter and view detailed game info

---

## 🧱 Tech Stack

### Frontend

- **React.js**
- **Redux Toolkit** (state management)
- **Tailwind CSS + Bootstrap** (styling)
- **React Router DOM** (routing)
- **Axios** (API requests)

### Backend

- **Node.js**, **Express.js**
- **MongoDB** (data storage)
- **JWT** (authentication & authorization)
- **Cloudinary** _(optional, for image storage)_

### APIs Used

Public APIs for:

- News
- Weather
- Cricket Data
- Movie/TV Series Metadata
- Games

## 🚀 Getting Started (Local Setup)

1. **Clone the repository:**

```
git clone https://github.com/AakashGaur03/MultiVerseHub.git
cd multiversehub
```

2. **Start the Frontend:**

```
cd Frontend
npm install
npm run dev
```

3. **Start the Backend:**

```
cd Backend
npm install
npm run dev
```
